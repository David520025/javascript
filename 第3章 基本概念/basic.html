  • 语法
    1. 区分大小写---ECMAScript中的一切（变量、函数名和操作符）都区分大小写：变量名test和变量名Test分别表示两个不同的变量；函数名不能用typeof,因为它是一个关键字，但是typeOf则完全可以是个有效的函数名。
    2. 标识符- 定义：指变量、函数、属性的名字，或者函数的参数。
        标识符可以是按照下列格式规则组合起来的一个或多个字符：
      • 第一个字符必须是一个字母、下划线（—）或是一个美元符号（$）;
      • 其它字符可以是字母、下划线、美元符号或数字。
        • 惯例（非强制性，但是更规范）：标识符采用驼峰大小写格式。第一个字母小写，剩余的每个单词首字母大写。
    3. 注释：包括单行注释和块级注释
    4. 严格模式 "use strict"
    5. 语句，ECMAscript 中的语句以分号“；”结尾，虽然不是必须的但是不要省略它。
      • 把多条语句组合到一个代码块中，代码块以左花括号（{）开头，右花括号（}）结尾：
        • if (test) {
        test = false
        alert (test) ;
        }
    6. 关键字和保留字（见P21）
    7. 变量：一个用于保存值得占位符。
      • var 是一个关键字，后接一个变量名（即一个标识符）
      • ECMAScript的变量是松散类型，即可以保存任何类型的数据
      • var message ; 像这样子未经初始化的变量，会保存一个特殊的值 undefined。 也可以直接初始化的同时进行赋值： var message = "hi" ;
      • var 操作符定义的变量将成为定义该变量的作用域中的局部变量。如果在函数中定义一个变量，这个变量在函数退出后就被销毁哦！
      • message = "hi"; // 全局变量 省略var后就定义了全局变量。
      • 可以使用一条语句定义多个变量，把每个变量用逗号 ， 分开即可： var message = "hi" , found = false , age = 29 ;
  • 数据类型
      ECMAScript中有五种基本数据类型:Undefined、Null、Boolean、Number、string。一种复杂数据类型Object,它本质是由一组无序的名值对组成。
    1. typeof 操作符
        对一个值使用typeof操作符可能返回下列某个字符串
      • "Undefined"---如果这个值未定义
      • "Boolean"---如果这个值是布尔值
      • "string"---如果这个值是字符串
      • "Number"---如果这个值是数值
      • "Object"---如果这个值是对象或Null
      • "function"---如果这个值是函数
