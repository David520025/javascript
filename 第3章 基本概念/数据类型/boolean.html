1. Boolean类型只有两个字面值：true和false.
  2. ECMAScript中所有类型的值都有与这两个Boolean值等价的值。要将一个值转化为其对应的Boolean值，可以调用转型函数Boolean(),如下所示：
    var message = "hello world";
    var messageAsBoolean = Boolean(message);//打印出来的值是 true
  3. true和false这两个值与数字值不是一回事，因此，true不一定等于1，而false也不一定等于0.规则详情见p27,这些规则对理解流控制语句很重要。
