String 类型

用于表示由0或多个16位Unicode字符组成的字符序列，即字符串。单引号和双引号成对使用即可，解释方式是相同的。
一个空格也是一个字符
  1. 字符字面量：也叫转义序列
  2. 字符串的特点：不可变的
  3. 转换为字符串 toString()方法-（几乎每个字符串都有一个toString()方法，但null和undefined没有）
  4. 在调用数值的toString()方法时，可以传递一个参数，输出数值的基数。
    var num = 10 ;
    console.log(num.toString()); // "10"
    console.log(num.toString(2)); // "1010"
    console.log(num.toString(8)); // "12"
  5. 转型函数String(),能将任何类型的值转换为字符串。
      遵循以下规则
    • 如果该值有toString()方法，则调用toString()方法
    • 如果值是null ,返回 “null”
    • 如果值是undefined ,返回 “undefined”
