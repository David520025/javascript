小结

  1. 像HTML中插入script的主要方法就是使用script元素
      script在HTML4.01定义了下列六个属性
    • async:可选
    • charset:
    • defer
    • langage:已废弃
    • src
    • type
  2. （曾经弄错过的）指定函数名称后要紧跟“（）”
  3. 在使用script元素嵌入script代码时，只需指定type属性，同本例子；嵌入式的时候在任何位置遇到script的结束标签都会终止，所以再加上一个"\"就可以了
  4. 带有src属性后的script标签之间的脚步文件会被忽略，只会下载外部的脚本文件
  5. script标签的位置放在head里面，浏览器就会先把脚本文件解析和执行完成后在碰到body元素；所以一般把脚本文件放在body元素最后一行。
  6. 延迟脚步就是在script中设置 defer="defer"，脚步就会等到整个页面解析完后再运行。相当于 立即下载，延迟执行。
  7. 异步脚步，类似于defer,async只适用于外部脚本文件，立即下载。 不同的是，多个异步脚步文件在一块时，并不会保证按照指定他们的先后顺序执行。
  8. noscript元素：用以在不支持script的浏览器中显示替代的内容
    • 浏览器不支持脚步
    • 浏览器支持脚步，但是脚本被禁用
      • 只有符合至少一个条件，noscript中的内容才会显示出来。
